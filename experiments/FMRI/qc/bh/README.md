# Breath-Hold CVR QC Task

Psychtoolbox stimulus code for the breath-hold cerebrovascular reactivity task.

Timing:

- 4 s breathe in
- 16 s hold
- 40 s normal breathing
- 4 cycles
- 20 s final normal breathing

Total planned design duration: 260 s.

Run `main` from MATLAB. In debug mode, press `5` or `t` to start. Press `ESC` or `q` to abort.
